P2P website hosting

Censorship-Resistant P2P network that uses Bitcoin cryptography and Bittorrent technology to host dynamic websites.

Download View slideshow about ZeroNet
Completely free and open source.

Decentralizing the power

After a site is published on ZeroNet there is no way to take it down. Sites are hosted by any visitor who wishes to.
It's nowhere because it's everywhere.

The Offline Internet

You can host any site you like, this way you can access the site even while you are offline (zero-latency).
When a site is updated you will automatically receive all incremental updates done to the site.

Bitcoin-Grade Security

Password-less BIP32 based authorization: Your account is protected by same cryptography as your bitcoin wallet.

Features

  • Tor network support
  • Easy, zero-configuration setup
  • Namecoin .bit domain support
  • Built-in SQL database with P2P data syncronization
  • Automatic, uPnP port opening
  • Multi-user sites
  • Real-time site update
  • Works on any browser/OS
